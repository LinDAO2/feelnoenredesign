# feelnoenredesign

Redesign of [feelnoen.com](https://feelnoen.com) — Noén, a luxury intimacy body-oil brand. Hero product: **The Nile Oil** (Blue Lotus).

## Stack
- Static starter (`index.html`) — to be expanded.
- Deployed via Vercel.

## Develop
Open `index.html` in a browser, or run any static server:

```bash
npx serve .
```
